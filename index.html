Not Enough Object Info

Detailed, real-time information about blocks, entities, and items—right on your HUD and in tooltips.

Explore Features Installation View on GitHub
Mod ID: notenoughobjectinfo Author: EmanuelPlays

✦ Features

HUD overlay and enhanced item tooltips designed for technical players, pack makers, and modded power users.

HUD Overlay

Look at any block, entity, or held item to see rich, contextual information.

Category Information Shown
Blocks Name, Registry ID, Mod, Hardness, Blast Resistance, Harvest Tool, Block/Sky Light, Redstone Power, Flammability, Friction, Block State Properties, Tags, Block Entity type + NBT
Entities Name, Registry ID, Mod, Category, colour-coded Health bar, Max Health, Armor, Movement Speed, Attack Damage, Active Potion Effects with duration, Position, Dimension, Tags, NBT
Held Item Registry ID, Mod, colour-coded Durability bar, Enchantability, Burn Time, Food Value + Saturation, Max Stack Size, Tags

Item Tooltips

Every item tooltip is upgraded with technical metadata and survival-relevant stats.

  • Registry ID & source Mod
  • Remaining Durability (%)
  • Enchantability
  • Max Stack Size
  • Burn Time (ticks + seconds)
  • Food Value & Saturation
  • Repair ingredient info
  • NBT summary
  • Item tags

⌨ Keybindings

All keybindings are configurable in Options → Controls → Not Enough Object Info.

Key Action
H Toggle HUD on/off
J Cycle display mode (COMPACT → NORMAL → VERBOSE)
N Toggle NBT display

📖 Display Modes

Choose how much information you want to see at a glance.

COMPACT

Name, ID, Mod, and a few key stats only—perfect for minimal HUD setups.

NORMAL (default)

Full standard info without the most verbose fields. Balanced for everyday play.

VERBOSE

Everything: state properties, tags, NBT, and all attributes for deep debugging.

⚙ Configuration

Fine-tune every aspect of the overlay and tooltip behaviour.

Accessing the config

  • Via Mod Menu → NEOI → Config button
  • Or edit config/neoi-client.toml in your Minecraft instance folder

Options

  • ~50 individual display toggles
  • HUD position (X/Y)
  • Text scale (25–200%)
  • Background transparency
  • Text shadow on/off

🔨 Building from Source

Requires JDK 17 and an internet connection for Forge MDK dependencies.

Gradle commands


# Clone / place the project at your desired path
cd NotEnoughObjectInfo

# Generate IDE run configurations (optional)
./gradlew genIntellijRuns   # IntelliJ IDEA
./gradlew genEclipseRuns    # Eclipse

# Build the mod jar
./gradlew build
# Output: build/libs/NotEnoughObjectInfo-1.0.0.jar

Tips

  • Use a dedicated Gradle user home for modding projects.
  • Run ./gradlew --refresh-dependencies if something looks out of date.
  • Commit gradle.properties and build.gradle, but not .gradle/.

📦 Installation

Get NEOI running in your modded Minecraft instance.

  1. Install Minecraft Forge 1.20.1-47.2.0 (or newer 47.x).
  2. Drop NotEnoughObjectInfo-1.0.0.jar into your mods/ folder.
  3. Launch the game—NEOI is ready!
