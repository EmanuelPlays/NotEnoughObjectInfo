# Security Policy

## Supported Versions

Only the latest release of **Not Enough Object Info** receives security updates. Older versions are not actively maintained.

| Version | Supported          |
| ------- | ------------------ |
| 2.x | ✅ Yes                 |
| 1.x     | ✅ Yes            |
| < 1.0   | ❌ No             |  

---

## Reporting a Vulnerability

If you discover a security vulnerability in Not Enough Object Info, **please do not open a public GitHub Issue.** Public disclosure before a fix is available puts other users at risk.

### How to Report

1. **Go to the [Security Advisories](../../security/advisories/new) tab** of this repository and open a private advisory, **or**
2. **Open a ticket on our discord server:** 

### What to Include

To help investigate and resolve the issue as quickly as possible, please provide:

- A clear description of the vulnerability and its potential impact
- The version of the mod affected
- Step-by-step instructions to reproduce the issue
- Any relevant logs, screenshots, or proof-of-concept code
- Your Minecraft/Forge version and environment details

---

## What Happens Next

| Timeline | Action |
|----------|--------|
| **Within 48 hours** | Acknowledgement of your report |
| **Within 7 days** | Initial assessment and severity classification |
| **Within 30 days** | A patch or mitigation will be released (for valid, high-severity issues) |

You will be kept informed throughout the process. Once a fix is released, you will be credited in the changelog unless you prefer to remain anonymous.

---

## Scope

This policy covers the **Not Enough Object Info mod JAR and its source code** hosted in this repository.

The following are **out of scope:**

- Vulnerabilities in Minecraft itself or Forge
- Issues in third-party mods or libraries used alongside this mod
- Bugs that do not have a meaningful security impact (use the regular Issue tracker for those)

---

## Responsible Disclosure

We follow a **coordinated disclosure** approach. Please give us a reasonable amount of time to investigate and release a fix before disclosing the issue publicly. We commit to acting promptly and transparently throughout the process.

Thank you for helping keep Not Enough Object Info safe for everyone. 🛡️
